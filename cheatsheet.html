CAPTURE Task

Trial-by-Trial (Long Format) Data Schema Cheatsheet

📊 Participant & Metadata Columns

These columns are appended to every row in the dataset to simplify multi-subject analysis and database merging.

Column Header Data Type Description & Source
participant_id String Unique identifier for the participant. Entered on the first setup screen.
session_date String The date of the session (YYYY-MM-DD format). Selected on the second setup screen.
ra_name String The name of the Research Assistant running the test. Entered on the third setup screen.

⏱️ Standard jsPsych Temporal & Event Columns

Standard columns generated automatically by jsPsych to track chronology, durations, stimuli, and keystrokes.

Column Header Data Type Description & Source
trial_index Integer Sequential index tracking the exact chronological order of screens/events (starts at 0).
trial_type String The plugin module name (e.g. participant-id-input, recognition-task, timeline-task).
time_elapsed Integer Cumulative milliseconds from the very start of the experiment until this trial completed.
rt Integer Reaction time (milliseconds) from screen load to participant submission/click.
stimulus String The text shown to the user or the local asset file path of the artwork presented.
response String/Int The raw button selection, text entered, or placement coordinates/seconds submitted.

🧠 Custom Experimental Columns (Conditional)

Dynamic columns populated depending on the trial type. Rows belonging to other trials will show empty/null values here.

Recognition Tasks (trial_type: recognition-task)

Column Header Data Type Description & Source
image_id String The local path of the artwork image.
recognized Boolean true if recognized ("Definitely yes" / "Maybe yes"); false if unrecognized.
recognition_confidence String Exact rating value: "Definitely yes", "Maybe yes", "Maybe no", or "Definitely no".
relative_duration String Observation time rating relative to the average (e.g. "Much less", "About the same", "Much more").
rt_recognition Integer Reaction time (ms) for the initial recognition choice screen.
rt_phase2 Integer Reaction time (ms) for the confidence and duration rating screen (if recognized).

Timeline Tasks (trial_type: timeline-task)

Column Header Data Type Description & Source
timeline_position_sec Integer Relative time placement on the 45-minute gallery timeline, snapped to seconds.
estimated_duration_sec Integer The estimated duration of the event (defaults to null).
rt_timeline Integer Reaction time (ms) from timeline screen onset to the placement click.

Cued Recall Tasks (trial_type: cued-recall-task)

Column Header Data Type Description & Source
artwork_index Integer The list order index (0 to 4) mapping back to the chronological artwork sequence.
cued_recall_response String The participant's typed descriptions or descriptions of what they recall about the artwork.
